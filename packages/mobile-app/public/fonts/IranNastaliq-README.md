# IranNastaliq Font Required

This directory needs the IranNastaliq font files for the splash screen Persian tagline.

## Required Files
- `IranNastaliq.woff2` (primary)
- `IranNastaliq.woff` (fallback)

## How to Obtain
1. Download from the temp branch: https://github.com/atomicdeploy/yektayar/blob/temp/IranNastaliq-Font.zip
2. Extract the zip file
3. Place the `.woff2` and `.woff` files in this directory
4. Delete this README file

## Alternative Source
If the temp branch is unavailable, download from:
https://github.com/rastikerdar/iran-nastaliq-font
