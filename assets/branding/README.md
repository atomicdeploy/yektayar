## For Developers

When implementing features:

1. **Always use brand colors** from the color palette
2. **Check theme variables** in:
   - Mobile: `packages/mobile-app/src/theme/variables.css`
   - Admin: `packages/admin-panel/src/assets/main.css`
3. **Follow spacing guidelines** (8px base unit)
4. **Test accessibility** (WCAG AA minimum)
5. **Ensure consistency** across platforms

## For Designers

When creating designs:

1. **Use official colors** - no custom variations
2. **Maintain logo integrity** - proper spacing and sizing
3. **Follow typography hierarchy** - see BRANDING-GUIDE.md
4. **Design for accessibility** - check contrast ratios
5. **Consider both light and dark modes**
